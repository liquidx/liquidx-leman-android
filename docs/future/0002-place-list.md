Create a new suite of features that have to do with creating, editing and maintaining lists of places. Each place could be added through searching or from check-ins.

We should be able to assign an emoji to the list, a tag to the list (maybe type of lists, favorites, want to go, planned, and also th region/country/city). We should have a page that shows the list with a map and the places below. We should be able to export and import these lists, so some way import and export format. When a place is added to the list, we also should have a description or note, maybe also formatted as tags like "recommended by", "type of place".

When browsing the list, we should be able to see what lists that this place is also part of when we browse, so we can see which other lists it is of.
